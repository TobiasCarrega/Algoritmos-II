# Algoritmos II

Repositorio destinado a almacenar las actividades y ejercicios realizados durante la cursada de **Algoritmos II**.

## Objetivo

En este repositorio se irán subiendo las actividades correspondientes a cada clase, con el fin de:

- Registrar el progreso del curso.
- Mantener organizado el material trabajado.
- Facilitar la revisión y el aprendizaje colaborativo.

Las actividades estarán organizadas por **clase o temática**, permitiendo un seguimiento claro del contenido visto durante la materia.
